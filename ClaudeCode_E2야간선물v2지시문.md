## 1. [1순위] 코스피200 야간선물 직접수집 시도 (KRX OPEN API)

### 1-A. KRX OPEN API 키 확인
- `.env`에서 KRX OPEN API 인증키(`KRX_OPENAPI_KEY` 또는 기존 키명) 읽기.
- **키 없으면**: 발급 안내만 남기고(이주혁이 openapi.krx.co.kr에서 발급) **1순위는 "키 미설정으로 보류"**, 곧장 2순위(미국선물 폴백)로. ClaudeCode가 키 발급·입력 금지.
- 키 있으면 1-B 진행.

### 1-B. 야간선물 데이터 존재 여부 실호출 확인
- KRX OPEN API의 **파생상품(선물) 관련 엔드포인트**를 조사·호출하여 **코스피200 야간선물(야간세션) 데이터가 제공되는지** 확인.
  - 정규(주간)선물만 있는지, 야간세션이 별도로 있는지.
  - 야간선물 종가/등락이 잡히면 → 함수 `get_kospi_night_futures()` 구현(종가, 전일정규대비 변화, 시각).
  - **안 잡히면**(정규선물만, 야간 미제공): "KRX OPEN API 야간선물 미제공" 정직하게 결론 + 2순위 폴백.
- 실패/불명확 시 추정 없이 결과 명시. 무리하게 "있는 것처럼" 만들지 말 것.

## 2. [2순위 폴백] 미국선물 프록시 (1순위 막힐 때)

> 1순위가 되면 이건 보조 참고. 1순위가 막히면 이게 메인.

- `get_us_futures()`: yfinance `ES=F`(S&P500선물)·`NQ=F`(나스닥선물) 현재값+미국 정규장종가 대비 변화%+시각. 실패 시 None+log. timeout=10.

## 3. 방향성 참고신호 (소스 무관 공통)

- `assess_kospi_gap_signal()`:
  - **1순위 가능 시**: 코스피 야간선물 등락을 **직접** 방향성 신호로(가장 직접적). "야간선물 +X% → 시초가 갭업 가능성(참고)".
  - **폴백 시**: 미국선물(ES=F/NQ=F, 한국 기술주 비중 고려해 NQ 가중) 변화율로 방향성 추정. "미국선물 기준 참고".
  - **철학(필수)**: 단정 금지. "갭업 **가능성**(참고용, 아침 수급·뉴스로 변동 가능)" 수준. 임계값(±0.5% 등) 잠정·사후검증 조정 명시.
  - 시간대 인지: 한국 야간(18시~새벽)에 의미. 낮·주말·휴장이면 "해당없음".
  - 출력에 **어느 소스 기준인지 표기**(코스피 야간선물 직접 / 미국선물 프록시).

## 4. 한국 브리핑 연결

- `format_night_futures_section()`: E1 패턴으로 한국 브리핑("오늘 한국시장 어때?")에 append만 하면 끼워지는 텍스트블록 반환. 브리핑 본체 최소 수정.
- 출력 예(1순위): "🌙 코스피200 야간선물(직접): +0.4% → 시초가 갭업 가능성(참고용)".
- 출력 예(폴백): "🌙 야간 미국선물(프록시): S&P500선물 +0.3%·나스닥선물 +0.5% → 코스피 갭업 가능성(참고용, 코스피 야간선물 직접수집은 KRX API 미제공/키미설정)".
