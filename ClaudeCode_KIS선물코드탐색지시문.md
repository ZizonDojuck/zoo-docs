## 2. [핵심] 코스피200 선물 종목코드 자동 탐색

> 추측 금지. KIS API로 "현재 거래되는 코스피200 선물 종목 리스트"를 받아 근월물 코드를 확정.

### [2-A] 선물 종목 마스터/리스트 조회 API 탐색
- KIS 국내선물옵션 카테고리에서 **선물 종목 리스트/종목 검색/상품기본정보** 류 엔드포인트를 공식 문서/GitHub(`koreainvestment/open-trading-api`의 `domestic_futureoption`) 기준으로 찾아 호출.
  - 후보: 선물옵션 종목 리스트 조회, 상품기본정보 조회, 또는 지수선물 종목 검색 API.
  - 코스피200 선물(기초자산=KOSPI200, 상품유형=지수선물)의 **현재 거래 가능한 월물 종목코드**(9자리)를 받아온다.
- 받은 코드 중 **근월물**(만기 가장 가까운 것)을 선택.
- **야간거래 종목코드가 주간과 별개인지** 확인(KRX 야간선물 EP가 별도 tr_id/코드 쓸 수 있음).

### [2-B] 종목 마스터 파일 대안 (API로 안 되면)
- KIS는 종목코드 마스터를 **다운로드 파일**(예: 선물옵션 종목마스터)로 제공하기도 함. API 리스트 조회가 안 되면 이 방법 조사(단 자동 다운로드 가능할 때만, 무리한 스크래핑 금지).
- 그래도 안 되면: "API로 코드 자동확정 실패" 정직하게 보고 → 이주혁이 HTS/MTS에서 직접 확인하는 방법으로 폴백(KIS_K200_FUT_CODE 수동주입).
