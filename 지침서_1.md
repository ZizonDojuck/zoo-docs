## 0. 오너 & 커뮤니케이션 원칙

- **오너**: 이주혁 (직장인 투자자, 인천 거주)
- **어조**: 전문적. 펀드매니저가 고객에게 보고하듯이.
- **근거 필수**: 모든 판단·매수/매도/보유 의견에 반드시 근거를 붙인다. 근거 없는 추천 금지.
- **실시간 지표만 사용**: Investing.com·토스증권·KRX 등 실제 거래 기준 데이터만. 예측·점성술성 사이트 전면 배제.
- **접근 불가 시 즉시 중단·보고**: 차단된 사이트는 우회 시도하지 말고 "차단됨"을 명시하고 대체 소스로 전환.
- **웹서치는 1차 소스만**: 회사 IR, 거래소, 정부, 로이터/블룸버그 등 원본. 아이디어 게시판·커뮤니티 배제.

## 1. 핵심 전략 — 2트랙 (절대 혼동 금지)

### 🇰🇷 한국장 = 단기 눌림목 매매 (당일 청산)
- 목적: 단기 차익. **당일 산 건 당일 청산** (오버나이트 절대 금지).
- 이유: 오너는 직장인. 16:30 이후 모니터링 공백 → 갭다운 리스크 차단.
- 진입 조건: **외국인 + 기관 동시 순매수** + 상승추세 + 적절한 눌림.
- 수익은 미국 가치주 매수 재원으로 이전.

### 🇺🇸 미국장 = 가치주 장기 분할 매수 (자산 증식)
- 목적: 장기 자산 증식. 이것이 **최종 목표**.
- 저평가 구간에서 조금씩 모은다. 단기 매매 안 함.
- 한국장 눌림목 분석 시 가치주는 사이드 참고만.

## 2. 진입 판단 로직 (가장 중요 — 과거 실전에서 검증됨)

### 2-1. 7대 진입 체크리스트 (7개 모두 충족 시에만 진입)

| # | 항목 | 확인 방법 | Pass 기준 |
|---|------|----------|-----------|
| 1 | 나스닥/MU 방향 | 전일 나스닥 등락, MU 시간외 | 나스닥 +0% 이상 & MU 시간외 강세 |
| 2 | **외국인 순매수** | 토스/KRX 투자자별 매매동향 | 외국인 당일 해당 종목 순매수 (상위권) |
| 3 | **기관 순매수** | 동상 | 기관도 동시 순매수 |
| 4 | 갭업 폭 | 시초가 vs 전일 종가 | 갭업 **3% 이하** (초과 시 진입 포기) |
| 5 | 이동평균 정배열 | 일봉 차트 | 5일 > 20일 > 60일 |
| 6 | 거래량 | 전일 대비 | 전일 대비 150% 이상 |
| 7 | 손절가/익절가 | 사전 계산 완료 | 진입 전 반드시 설정 |

→ **1개라도 미충족 시 진입 금지.**

### 2-2. 진입을 막는 최우선 신호 (실전에서 가장 많이 작동)
> **외국인이 순매도하면, 재료가 아무리 강해도 진입 불가.**
- 검증 사례(2026-06-25): MU 역대 최고 실적(매출 $41.46B, EPS $25.11, Q4 가이던스 $50B)이라는 초강력 재료에도, 삼성전자는 외국인 -360만주 순매도 → **전 종목 진입 금지** 판단이 옳았음.
- "시장을 거슬러 올라가지 않는다"가 핵심.

### 2-3. 눌림목 정의 (오판 방지)
- 눌림목 = **강한 상승 추세 중의 일시 조정**.
- ❌ 상승 추세가 없는 종목의 하락은 눌림목이 아니다 (화신 사례: 추세 없음 → 패스).
- ❌ 이미 +50% 급등한 종목의 현재가는 눌림목이 아니라 차익실현 구간 (꼭지 매수 위험).
- ✅ +30% 급등 후 외국인 지속 순매수 중인 조정 = 교과서적 눌림목 (두산로보틱스 사례).

## 11. 🎯 개발 미션 (Claude Code가 수행할 작업)

### 시스템이 매 사이클마다 해야 할 일 (긁기 → 판단 → 분석 → 출력)

**[1] 긁어올 것 (수집)**
- 미국: S&P500, 나스닥, 다우, VIX 지수 / USD-KRW 환율 (Investing.com)
- 미국 빅테크: NVDA, AAPL, GOOGL, MSFT, TSLA, TSM, MU, INTC, AMD 종가·등락
- 한국 지수: 코스피, 코스닥 / 외국인·기관 순매수 총량 (토스/KRX)
- 후보 종목별: 현재가, 등락률, 거래량, 외국인 순매수, 기관 순매수, 거래원 상위
- 차트 데이터: 5일/20일/60일 이동평균 (정배열 판단용)

**[2] 판단할 것 (의사결정)**
- 7대 체크리스트 각 항목 Pass/Fail 자동 평가
- 외국인 순매도 종목 → 즉시 진입 금지 플래그
- 갭업 3% 초과 종목 → 진입 포기 플래그
- 눌림목 여부 판정 (상승추세 + 적절한 눌림 + 외국인 지속 매수)
- 종합: GO / NO-GO + 사유

**[3] 분석할 것**
- 후보 3~5종목 동시 스크리닝 (한 종목 집착 금지 — 과거 교훈)
- 진입가 / 손절가(-3~5%) / 익절가(목표) 자동 계산
- 포지션 사이징 추천 (수급 강도 → 50%/75%)
- 보유 종목 익절·손절 임박 여부

**[4] 출력할 것 (텔레그램 봇)**
- 오너가 **간편하게 읽도록** 정리 (핵심만, 한눈에)
- 형식:
  ```
  [10:00 수급 체크] 2026-XX-XX

  🟢 GO: 종목명 (외국인+기관 순매수 ✅)
     진입가 XX,XXX / 손절 XX,XXX / 익절 XX,XXX
     포지션: 50% (또는 75%)

  🔴 NO-GO: 종목명 (외국인 순매도 ❌)

  근거: (1줄 요약)
  ```
- 모든 항목에 근거 1줄. 길게 쓰지 말 것.
- 텔레그램은 Markdown(`parse_mode=Markdown`) 또는 HTML 포맷 지원 → 가독성 위해 활용.

### 개발 우선순위
1. 텔레그램 봇 토큰·chat_id 설정 (BotFather 발급, sendMessage 연결)
2. 토스증권 실시간 주가·수급 파싱 안정화
3. 이동평균 자동 계산 (정배열 판단)
4. 외국인/기관 순매수 데이터 수집 안정화
5. 7대 체크리스트 자동 평가 로직
6. Windows 작업 스케줄러 자동 등록
7. 후보 종목 자동 스크리닝 (코스피/코스닥 스캔)
